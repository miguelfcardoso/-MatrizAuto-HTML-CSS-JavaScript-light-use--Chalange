## üß© Project Overview

This repository showcases a responsive, visually rich layout for **MatrizAuto**, a car shopping / auto-related interface.  
The goal was to replicate a design (images, layout, responsive behavior) using only HTML, CSS, and minimal JS for interactivity.

Key aspects:
- Pixel-perfect layout (or as close as possible)  
- Responsive design (desktop / mobile)  
- Visual components: header, menu, carousels, images, footers  
- Basic interactivity (menu toggles, etc.)
